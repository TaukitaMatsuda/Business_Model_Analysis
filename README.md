# Анализ бизнес-модели магазина экотоваров

### О проекте
Исследование операционной эффективности и юнит-экономики онлайн-магазина экотоваров в связи с планами по масштабированию бизнеса в регионы. В рамках проекта проведен глубокий анализ когорт, расчет метрик окупаемости и оценка эффективности рекламных каналов.

### Цели
1. Оценить текущую рентабельность бизнес-модели.
2. Определить наиболее эффективные регионы и каналы привлечения (SEO, CPC, SMM).
3. Проанализировать удержание клиентов (Retention) и окупаемость инвестиций (ROMI).

### Стек технологий
* **Google Sheets / Excel:** Сложные формулы, сводные таблицы, когортный анализ.
* **Аналитические метрики:** CAC, LTV, ROMI, Retention, ARPU, COGS.

### Ключевые выводы (Insights)
* **Эффективность каналов:** SEO показал самый высокий ROMI (375%), в то время как SMM является самым дорогим каналом с низкой окупаемостью в ряде регионов.
* **География:** Москва и Санкт-Петербург — лидеры по прибыли. Волгоград имеет потенциал для роста при оптимизации рекламных затрат.
* **Удержание:** Выявлен тренд на снижение Retention во второй месяц жизни когорты, что требует внедрения программы лояльности для "домашних расходников".

### Структура проекта
* 1. Возвращаемость клиентов для каждой когорты				
* 2. Показатели юнит-экономики в разрезе их недельных сроков жизни				
* 3. Показатели юнит-экономики по когортам, агрегированные за весь срок их жизни				
* 4. Показатели юнит-экономики для каждого региона				
* 5. Возвращаемость инвестиций. Самые эффективные источники привлечения. Самые эффективные регионы				
* 6. Расчет юнит метрик для когорт				
* 7. Расчет юнит метрик для регионов				
* 8. Выводы				
* 9.Исходные данные				
* 9.1.Таблица user_log				
* 9.2 Таблица users				
* 10. Справочник
